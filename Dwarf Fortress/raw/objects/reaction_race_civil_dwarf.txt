# Merchant Building Reactions:
# Merchant Commodities Building:
# A - so I wanted a basic necessities of Dwarf Society building - the goal was
# to not go overboard with what was available to dwarfs from a vendor.  at the 
# same time provide at least a place to sell off excess unwanted stones, fish, 
# meat, wood, etc. and provide some basic return of those at a higher price.
# in otherwords this shop is not a 1 for 1 returner but a 2 to 1 return.
# so value of 100gc = 2x30 = 60 ; 500gc = 300 value
# Item------ value    sell# buy#  # at 100gc(# at 500gc)
# Meat ----- 2x2=4    15    7
# Fish ----- 2        30    15
# Cheese --- 10x2=20  3(15) 1(7)
# Wood ----- 3        20    10     <- going 1/2 # same as marble
# Thread --- 6x2=12   5(25) 2(12)
# Cloth ---- 7x2=14   4(21) 20(10)
# Leather -- 5x2=10   6(30) 3(15)
# Coke ----- 5x2=10   6(30) 3(15)
# Gypsum --- 3        20    10     <- going 1/2 # same as marble
# Marble --- 3x2=6	  10    5
# Stone ---- 3        20    10     <- going 1/2 # same as marble
# Gem ------ 5x2=10   6(30) 3(15)
# Copper --- 5x2=10   6(30) 3(15)
# Silver --- 5x10=50  1(6)  ?(3)   <- going with 500gc version.
# Gold ----- 5x30=150 ?(2)  ?(1)   <- ????? 1 bar makes 500gc. ouch.
# Iron ----- 5x10=50  1(6)  ?(3)   <- going with 500gc version.
